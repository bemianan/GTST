**While typing a command (before pressing Enter), you can use keyboard shortcuts: 

    Ctrl + _ (Control and underscore) or Ctrl + /: Undoes the last edit to the current command        line. You can press it multiple times to undo consecutive changes.
    Ctrl + U: Erases everything from the current cursor position to the beginning of the line.
    Ctrl + W: Erases the last word before the cursor.
    Ctrl + Y: Pastes (yanks) the text that was most recently deleted using Ctrl + U, Ctrl + W, or     Ctrl + K.
    ctrl + k: Erases everything from the current cursor position to the end of the line
  
** Ctrl z equivalent on nano == alt + u
** In regular expressions, square brackets define a character class. By wrapping the entire expression in them, you are telling the script to match any single character contained within the brackets (like a literal ., *, @, g, etc.), rather than matching the sequence as a whole.*
